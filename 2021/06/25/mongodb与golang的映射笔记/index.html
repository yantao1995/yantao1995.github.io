bson 源码

bson.D

  • document

表示有序bson文档，传参使用。顺序不能乱，比如筛选，分组，求和。

bson.E

  • element

表示Key,Value 属性。

bson.M

  • map

表示无序bson文档，map类型。

bson.A

  • array

表示有序bson数组。
